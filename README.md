# CNN for Text Classification in Tensorflow

A slightly simplified implementation of Kim's [Convolutional Neural Networks for Sentence Classification](http://arxiv.org/abs/1408.5882) paper in Tensorflow.

## Requirements

- Python 2.7
- Tensorflow
- Numpy

## Running

Print parameters:

```
./train.py --help
```

Train:

```
./train.py
```
